## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

## Features

- Add, edit and delete todos

## TECH STACK USED

- React Native Expo
- Tailwind CSS
- Expo Sqlite
